Manual for COGNA Editor

Concept

The COGNA Editor is a tool to easily create and edit COGNA networks with a GUI.

Overview

In the editor you can create networks, where you place neurons, interfaces and subnetworks, which you can connect with each other. Every element has a certain set of parameters, which can be edited.
